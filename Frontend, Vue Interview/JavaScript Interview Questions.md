# JavaScript Interview Questions

## What is prototype and prototype chain? 什么是原型、原型链？

原型：相当于一个模具，用来生产实例对象。

原型链：原型对象有个指针指向构造函数，实例对象又有一个指针指向原型对象，就形成了一条原型链，最终指向null。
