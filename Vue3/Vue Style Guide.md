# Style Guide

[Component name casing in templates](https://vuejs.org/v2/style-guide/#Component-name-casing-in-templates-strongly-recommended)

Good

```javaScript
<!-- In single-file components and string templates -->
<MyComponent/>
<!-- In DOM templates -->
<my-component></my-component>
```

OR

```javaScript
<!-- Everywhere -->
<my-component></my-component>
```
